##Lesson 01 - HTML Basics


###Learning Objectives

*	Recognize the different roles and responsibilities in web development. 

*	Articulate the progression of HTML to HTML5.*	Apply HTML tags ```<html><head><!DOCTYPE html><body><h1><h6>``` to a web page and experiment with html tags.*	Practice using front-end web development tools (sublime text, finder, chrome).
